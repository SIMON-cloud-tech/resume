How Loyalty DNA Was Built

Building Loyalty DNA started with identifying core needs through consultations with hotel owners in Kenya, including those at institutions like MMUST. I began with a simple prototype using HTML, CSS, and JavaScript for the frontend, ensuring it's lightweight and deployable on platforms like GitHub Pages for zero-cost hosting.

The backend logic was conceptualized around a user-friendly dashboard where customers log meals via a web interface or integrated WhatsApp bots. Using JavaScript for dynamic elements like form handling and reward tracking, I incorporated modular code for easy scalability. Security features, such as encrypted data storage (planned with localStorage for MVP), ensure user privacy. The system was iteratively tested with real users at Tips Caffe Hotel, refining features like the 'Eat & Earn' program based on feedback. This agile approach allowed me to solve real pain points efficiently, making it a robust tool for "tech solutions for restaurants."
